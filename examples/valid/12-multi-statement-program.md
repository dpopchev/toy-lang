# 12 — Multi-Statement Program

A complete program combining multiple `let` and `print` statements to demonstrate linear top-to-bottom execution.

## Source

```toylang
let a = 1 + 2
let b = 3 * 4
let c = 0
print a
print b
print c
```

## Expected Output

```
1 OR 2
3 AND 4
0
```

## Covers

- Multiple `letStmt` declarations
- Multiple `printStmt` outputs
- OR expression, AND expression, and bare zero in one program
- Linear execution order
